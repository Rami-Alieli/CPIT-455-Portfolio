📂 What Is This?
This is my personal engineering portfolio for CPIT-455 Software Engineering II, built under the HIMMA curriculum. It documents 16 weeks of learning — not as a summary of slides, but as proof of applied thinking.
Every evidence section answers one question: how do I know I understood this? Not "what did I learn" — but "what can I now do, design, or justify that I couldn't before."
The portfolio is a single self-contained `index.html` file. No frameworks, no build step — open it in any browser.
